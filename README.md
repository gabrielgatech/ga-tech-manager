# G.A Tech Manager

Sistema ERP para gerenciamento de assistência técnica de celulares, notebooks, tablets e computadores.

## Objetivo

Criar um sistema completo para gerenciar:

- Clientes
- Aparelhos
- Ordens de Serviço
- Estoque
- Catálogo Técnico
- Financeiro
- Relatórios
- Etiquetas
- Central Técnica

Desenvolvido por G.A Tech.
